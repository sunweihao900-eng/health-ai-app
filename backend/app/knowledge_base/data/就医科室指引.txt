就医科室选择指引科普

本指引帮助您了解不同科室的基本职能，以便更好地进行就医选择。
具体就医决策应综合考虑症状特点、当地医院设置等因素。

一、常见症状对应科室参考

症状：发热、咳嗽、流鼻涕、咽痛
推荐科室：内科 / 全科 / 呼吸内科（如症状持续或加重）

症状：头痛、头晕
推荐科室：
- 突发剧烈头痛 → 急诊（排除脑血管意外）
- 慢性反复头痛 → 神经内科
- 伴颈部僵硬 → 急诊

症状：胸痛、心慌、气短
推荐科室：
- 急性、剧烈胸痛 → 急诊（优先排除心梗）
- 慢性、活动后胸闷 → 心内科
- 焦虑相关心悸 → 心内科或心理科

症状：腹痛
推荐科室：
- 急性剧烈腹痛 → 急诊 / 普外科
- 慢性腹痛、消化不良 → 消化内科（胃肠科）
- 右下腹疼痛 → 急诊（排除阑尾炎）
- 女性下腹痛 → 妇科

症状：皮肤问题（皮疹、湿疹、瘙痒、痤疮）
推荐科室：皮肤科

症状：骨关节疼痛、肌肉疼痛
推荐科室：
- 外伤相关 → 骨科 / 急诊
- 慢性关节疼痛 → 骨科 / 风湿免疫科
- 颈肩腰痛 → 骨科 / 康复科

症状：眼睛不适（视力下降、眼红、眼痛）
推荐科室：眼科
注意：突发视力丧失 → 急诊眼科

症状：耳鼻喉问题（耳鸣、耳痛、鼻塞、声音嘶哑）
推荐科室：耳鼻咽喉科（ENT）

症状：泌尿系统（尿频、尿急、尿痛、血尿）
推荐科室：
- 感染症状 → 内科 / 泌尿外科
- 结石症状 → 泌尿外科

症状：妇科问题（月经异常、下腹痛、白带异常）
推荐科室：妇科

症状：儿童常见病
推荐科室：儿科（14岁以下建议选儿科）

症状：情绪问题、睡眠障碍、精神压力
推荐科室：精神科 / 心理科 / 心理援助（可先从心理咨询开始）

症状：口腔问题（牙痛、牙龈出血、口腔溃疡）
推荐科室：口腔科

二、就医类型选择：门诊 vs 急诊

选择急诊的情形：
- 症状突然加重且严重
- 剧烈疼痛难以忍受
- 出现意识改变、呼吸困难
- 外伤出血、骨折
- 中毒、药物过量
- 婴幼儿高热（超过39℃）

选择门诊的情形：
- 慢性病定期复诊
- 症状出现较长时间但无突然加重
- 需要专科检查或治疗

三、首次就诊准备建议

为了提高就诊效率，建议提前准备：
1. 整理症状特点：何时开始、部位、性质、加重/缓解因素
2. 携带既往病历和检查报告
3. 列出正在服用的药物
4. 记录过敏史
5. 准备想问医生的问题

四、互联网医院和线上问诊适用情形
适合线上问诊：
- 慢性病复诊、处方续开
- 轻微感冒、皮肤问题等常见病
- 检查报告咨询
不适合线上：急症、需要体格检查、初次就诊复杂病情

---
注意：本指引仅提供一般性参考，就医科室的最终选择应根据实际症状、
医院设置及挂号资源综合决定。如有疑问，可先咨询导诊台或全科医生。
