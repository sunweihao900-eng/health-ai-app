# 隐私政策

**健康AI咨询助手**
版本：1.0 | 生效日期：2026年3月20日

本隐私政策依据《个人信息保护法》（PIPL）、《数据安全法》及相关法规制定。

---

## 一、信息收集

### 1.1 我们收集的信息
| 信息类型 | 收集方式 | 用途 |
|---------|---------|------|
| 账户信息（用户名） | 用户注册时提供 | 身份认证、服务个性化 |
| 咨询内容（脱敏后） | 用户主动输入 | 提供AI咨询服务、服务改进 |
| 设备信息（设备型号、OS版本） | 自动采集 | 技术兼容性、安全分析 |
| 访问日志（IP地址、时间戳） | 自动记录 | 安全防护、问题排查 |

### 1.2 我们不收集的信息
- 真实姓名（输入内容中的姓名将被自动脱敏）
- 手机号、身份证号（输入内容中将被自动脱敏）
- 精确位置信息
- 生物特征信息

## 二、PII自动脱敏

根据《个人信息保护法》最小化原则，您输入的咨询内容在**传输前**将经过自动脱敏处理：

- 手机号 → 138****5678（保留前3位和后4位）
- 身份证号 → 全部遮盖（仅保留地区码前6位）
- 邮箱地址 → ab***@example.com
- 银行卡号 → 全部遮盖
- 识别到的姓名 → 李**

**脱敏处理在您的设备端和服务器端均会执行，原始含PII信息不会被存储。**

## 三、信息使用

我们使用收集的信息用于：
1. 提供AI健康咨询服务
2. 改进AI模型的服务质量（仅使用脱敏数据）
3. 合规审计和安全监控
4. 技术故障排查

**我们不会将您的信息用于：**
- 定向广告投放
- 出售给第三方
- 用于保险定价或医疗商业决策

## 四、信息存储与保留

| 数据类型 | 存储位置 | 保留期限 |
|---------|---------|---------|
| 脱敏对话日志 | 国内服务器（AES-256加密） | 90天后自动清除 |
| 账户信息 | 国内数据库 | 账户注销后30天内删除 |
| 访问日志 | 安全系统 | 6个月 |

所有数据存储于中国境内服务器，符合数据本地化要求。

## 五、信息安全措施

- **传输安全**：全程使用HTTPS/TLS 1.3加密传输
- **存储安全**：AES-256加密静态数据
- **访问控制**：最小权限原则，员工访问需审计日志
- **脱敏处理**：PII数据存储前强制脱敏
- **安全评估**：已通过等保二级评测（上线前完成）

## 六、您的权利

依据《个人信息保护法》，您享有以下权利：

1. **查阅权**：查看我们持有的您的个人信息
2. **更正权**：更正不准确的个人信息
3. **删除权**：请求删除您的个人信息
4. **撤回同意权**：随时撤回对信息处理的同意
5. **转移权**：将个人信息转移至其他服务商
6. **投诉权**：向网信办或个人信息保护机构投诉

**如需行使上述权利，请联系：privacy@example.com**
我们将在15个工作日内回复。

## 七、Cookie使用

Web版本使用Cookie用于：
- 维持登录状态（必要）
- 存储用户同意记录（必要）

我们不使用第三方追踪Cookie。

## 八、未成年人保护

本服务不向14周岁以下未成年人提供。若发现收集了未成年人信息，我们将立即删除相关数据。

## 九、隐私政策变更

本政策如有重大变更，将通过站内通知提前30天告知用户。继续使用服务视为接受更新后的隐私政策。
